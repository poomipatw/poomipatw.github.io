Pain Point

    Unpleasing UI Outdated UI and unable to deliver message fluently.
    Multi currency support: Previous ver. do not support multi currency exchange and the flow is not fluent.

Goal

Create up to date UI and elimiate back and forth flow shifting between each currency.

Existing Fx Flow

This existing flow may work for few currencies in the platform but it won't be a ble to handle more than 3 currencies.

In order to shift between currencies, user has to go way back to the entrance to change the currencies option, then choose either buy or sell.

Pain:
This may have slow down the transaction flow for trader who want to instantly buy stock from different markets.

New Fx Flow

All in one:
From a single entry point, all Fx function are available here. User can chose to buy, sell, and exchange multiple currencies within a single screen.

It works like an Fx booth!

This will help reduce the flow length and increase chance for user to buy their stock with the price they want.
